üí° Pro Tip: Gestire Pi√π Progetti con un Solo Database

Problema: Altervista offre un solo database gratuito, ma potresti voler sviluppare pi√π progetti.

Soluzione: Usa i prefissi delle tabelle per partizionare logicamente il database!

Strategia di Partizionamento

1. Per ogni progetto, usa un prefisso diverso per le tabelle:

-- Progetto Esercizi PHP (tabelle con prefisso esercizi_)
esercizi_studenti
esercizi_corsi
esercizi_voti

-- Progetto Blog (tabelle con prefisso blog_)
blog_posts
blog_comments
blog_users

-- Progetto E-commerce (tabelle con prefisso shop_)
shop_products
shop_orders
shop_customers

2. Nel codice, definisci il prefisso come costante:

// In config/db_config.php
define('DB_PREFIX', 'esercizi_');

// Quando crei query
$table = DB_PREFIX . 'studenti';  // Risultato: esercizi_studenti
$query = "SELECT * FROM $table";

3. Vantaggi di questo approccio:

  • ‚úÖ Un solo database Altervista per pi√π progetti
  • ‚úÖ Facile identificare le tabelle di ogni progetto
  • ‚úÖ Nessun conflitto tra progetti diversi
  • ‚úÖ Puoi fare backup selettivi per progetto
  • ‚úÖ Cambi progetto solo modificando DB_PREFIX
üí° Suggerimento: Mantieni una lista dei prefissi usati in un file di testo per non dimenticarli!

Problemi Comuni e Soluzioni

‚ùå Apache non si avvia (XAMPP)

Causa: Porta 80 occupata (Skype, IIS, altri programmi)

Soluzione:

  1. Chiudi Skype o programmi che usano porta 80
  2. OPPURE cambia porta Apache:
    • XAMPP Control Panel ‚Üí Config ‚Üí httpd.conf
    • Cerca Listen 80 ‚Üí cambia in Listen 8080
    • Cerca ServerName localhost:80 ‚Üí cambia in ServerName localhost:8080
    • Salva e riavvia Apache
    • Usa http://localhost:8080/...

‚ùå MySQL non si avvia (XAMPP)

Causa: Porta 3306 occupata o altro MySQL installato

Soluzione:

  1. Apri Gestione Attivit√† (Ctrl+Shift+Esc)

  3. Riavvia XAMPP
